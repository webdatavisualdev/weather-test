Difference between binary and text files?

  • Bits represent character in text but a custom data in binary.
  • Text less prone to get corrupt as changes reflect as soon as the file is opened and can easily be undone.
    Binary can easily get corrupted, even a single bit change may corrupt the file.
  • Text can store only plain text in a file and binary can store different types of data (image, audio, text) in a single file.
  • Text widely used file format and can be opened using any simple text editor.
    Binary developed especially for an application and may not be understood by other applications.
  • Mostly .txt and .rtf are used as extensions to text files and binary can have any application defined extension.
